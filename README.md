## Overview

Snake-Online-Game is a dynamic 2D implementation of the classic Snake game, designed to provide an engaging and responsive gameplay experience.

**Why Snake-Online-Game?**

This project combines fluid player interactions with efficient game state management to deliver a seamless gaming experience. The core features include:

- 🐍 **Player Movement Management:** Facilitates smooth and responsive character navigation, eliminating laggy controls.
- 🎮 **Game State Management:** Efficiently handles game state and interactions for a seamless gameplay experience.
- 🧪 **Unit Testing Integration:** Employs Google Test for robust validation of game logic, enhancing code reliability.
- 📦 **Dynamic Resource Management:** Centralizes resource handling, promoting modularity and maintainability.
- ⚡ **Real-time Updates and Rendering:** Ensures immediate feedback on player actions, enhancing engagement.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** CPP
- **Package Manager:** Cmake

### Installation

Build Snake-Online-Game from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/maximusmenendez/Snake-Online-Game
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd Snake-Online-Game
    ```

3. **Install the dependencies:**

**Using [cmake](https://isocpp.org/):**

```sh
❯ cmake . && make
```
**Using [cmake](https://isocpp.org/):**

```sh
❯ cmake . && make
```
